Introducción a LiteMD

LiteMD es el editor de Markdown premium que estabas esperando.

  • Scroll Sincronizado: Desplazamiento fluido y paralelo.
  • WYSIWYG y Dividido: Edita visualmente o en texto plano.
  • Exportación: Genera PDFs e HTML con formato editorial.

Rust Tauri v2 GitHub

Diseñado para la excelencia

LiteMD combina la velocidad de una aplicación nativa con el formateo enriquecido en tiempo real.

Editor Doble Modo

Escribe en modo WYSIWYG interactivo estilo Notion o en texto plano Markdown clásico con vista dividida en tiempo real.

Scroll Sincronizado

Desplázate de manera bidireccional y fluida. La previsualización de tu documento se mantiene alineada con el editor a 60fps.

Drag & Drop Directo

Arrastra cualquier archivo `.md` o `.txt` sobre la ventana para abrirlo al instante con un elegante overlay translúcido.

Autoguardado Opcional

Activa el guardado automático de fondo. Tu progreso se asegura en segundo plano mostrando el estado en la barra superior.

Copiado en Bloques de Código

Copia rápidamente los bloques de código formateados al portapapeles con un botón flotante y animación de confirmación.

Exportación Editorial

Exporta tus documentos a PDF (formato A4 libre de saltos huérfanos), HTML estructurado premium o texto plano en un clic.
