Designed for excellence

LiteMD combines the speed of a native application with real-time rich markdown formatting.

Dual-Mode Editor

Write in interactive WYSIWYG mode (Notion-style) or classic raw Markdown syntax with real-time split view.

Synchronized Scroll

Navigate both panels bidirectionally and seamlessly. The rendering preview stays aligned with the editor at a smooth 60fps.

Direct Drag & Drop

Drag any `.md` or `.txt` file onto the window to open it instantly with a beautiful glassmorphic overlay.

Optional Auto-Save

Toggle auto-save on. Your progress is saved automatically in the background, displaying real-time badges in the titlebar.

Code Block Copy Button

Quickly copy formatted code blocks to your clipboard with a floating button and active checkmark animations.

Editorial Exports

Export your notes to PDF (A4 margin styling free of orphaned headings), structured HTML templates, or plain text.
